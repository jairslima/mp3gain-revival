# Archive

This directory preserves the historical material recovered for the MP3Gain revival effort.

## Contents
- `source/`: original source archive as recovered
- `binaries/`: historical binary packages
- `translations/`: translated help packages recovered as ZIP archives
- `site/`: mirrored project pages and related reference material

## Handling Rules
- Do not treat this directory as the active source tree
- Preserve file names and contents unless there is a clear archival reason to change them
- Prefer adding new documentation alongside archived material instead of rewriting the original files
- Avoid recursive inspection here during routine coding tasks; use it only when historical context is required
